Div Racer Online

A multiplayer game where players have to capture a randomly positioned dot 6 times to win. The multiplayer aspect was built with socket.io to allow for real-time play. Most of the game logic is implemented in the server to keep every player synchronized and up to date with the most recent movement.

Dick Blick Art Supply Store

The purpose of this project was to redesigned the current Dick Blick online art store(https://www.dickblick.com/) to provide a better UI for their customers. As well as improving the checkout service. The prototype was built with EJS, PostgreSQL, and PassportJS. Each item in the database has a category and sub category column which helps in being able to display all the based on the categories displayed on the navbar.
